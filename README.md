# 解説
右から襲い来る敵をサイコロ銃で迎撃しよう。
画面上部のサイコロの目の数だけ弾が出る。同じ目で止めてしまうと弾詰まりを起こして死にます。
3回続けて別の目で止めるとリロードされ、全ての目が解放されます。
